1) What is the difference between null and undefined?
Answer: undefined mane holo variable declare kora hoyeche kintu value assign kora hoy ni. Null mane holo intetiinally value null (empty) set kora hoyeche.

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Answer: map() hocche ekta array method. map() hocche every element er upor loop chalay kono change er command deya hole change kore notun array return kore but ager array te kono kichu change kore na. forEach() o ekta array method. forEach() shudhu element er upor loop chalay kintu kono kichu return kore na.

3) What is the difference between == and ===?
Answer: == & === ei 2 tai value check korleo, == shudhu value check kore kintu === value check korar pashapashi type o check kore.

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Answer: Scope holo code er kon jayga theke kon variable access kora jabe sheta rule. JavaScript e 3 dhoroner Scope ache.
i) Global Scope: Variable jodi function er baire declare kora hoy tahole sheta hobe golobal variable. jekono jayga theke access kora jabe.
ii) Function Scope: Variable jodi function er bhitore declare kora hoy tahole oi variable k oi function er moddhei access kora jabe. Function er baire theke access korte gele error dekhabe.
iii) Block Scope: Block { } er moddhe declare kora variable. Shudhu block er moddhei valid thake.
